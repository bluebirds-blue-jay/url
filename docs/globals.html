ensureHTTPProtocol

  • ensureHTTPProtocol(url: string, useHTTPS?: boolean | undefined): any
    • Defined in src/utils/ensure-http-protocol.ts:28
    description

    Ensure that an HTTP protocol is present. Set useHTTPS to true to force HTTPS. Set it to false to force HTTP. Let it blank to keep the original protocol or just ensure HTTP.

    example
    URL.ensureHTTPProtocol('example.com'); // http://example.com
    example
    URL.ensureHTTPProtocol('example.com', true); // https://example.com
    example
    URL.ensureHTTPProtocol('https://example.com', false); // http://example.com
    example
    URL.ensureHTTPProtocol('http://example.com', true); // https://example.com

    Parameters

    • url: string
    • Default value useHTTPS: boolean | undefined = undefined

    Returns any
